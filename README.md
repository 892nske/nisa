# NISA対応のシミュレーション

## モデルポート
* GPIFポートを使う<br>
→日本株、外株、日本債、外債：配分比率はそれぞれ25％<br>
→リスク・リターンもGPIF公表値を使う<br>
https://www.gpif.go.jp/topics/Adoption%20of%20New%20Policy%20Portfolio_Jp_details.pdf

## シミュレーション過程
* 合計は、15年投資期間
* 1~5年目は、年度初から240万ずっと投入
* 6~15年目は、新たな投入なし、リバランスのみ
* 運用期間中、各資産の配分比率は不変とする
* リバランスは、年１回（？調整）行う
